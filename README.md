# صالون كالا — موقع تعريفي (Salon Kala)

موقع تسويقي ثابت (Static Site) لمشغل وسبا كالا للتزيين النسائي في الخبر، مبني بـ HTML/CSS/JS صرف بدون أي خطوة بناء، جاهز للنشر على GitHub Pages مباشرة.

## النشاط
- **الاسم:** مشغل وسبا كالا للتزيين النسائي · Salon Kala
- **التصنيف:** صالون تجميل وسبا نسائي
- **المدينة:** الخبر — حي السد، المنطقة الشرقية
- **التقييم:** 4.6 على خرائط قوقل (220 مراجعة)

## سبب الاختيار
نشاط نسائي راقٍ بحضور جيّد على خرائط قوقل وصور حقيقية من داخل المكان، لكنه بلا موقع إلكتروني — فرصة واضحة لإطلالة رقمية تليق به وتسهّل الحجز عبر واتساب.

## فكرة التصميم
اتجاه "العناية الطبيعية الهادئة / Minimal Wellness": لوحة ألوان من الزيتي (#7E8B6B) والبيج الدافئ (#CDBBA4) والكريمي الناعم (#F5F1E9) مع نص حجري عميق. مساحات بيضاء واسعة، إيقاع هادئ، لمسات نباتية، وحركات ظهور لطيفة. الخط العنواني "El Messiri" والنص "Tajawal".

## التقنيات
- HTML5 + CSS3 + JavaScript vanilla (بدون أي مكتبات خارجية: لا React، لا jQuery).
- خطوط Google عبر `<link>` مع `preconnect`.
- IntersectionObserver لحركات الظهور مع fallback، Lightbox خفيف للمعرض، وقائمة جوال بملء الشاشة.
- بيانات JSON-LD من نوع `BeautySalon` مع التقييم الحقيقي.

## النموذج (Booking form)
- **حالة الـ backend: غير مربوط.** عند الإرسال يجهّز النموذج رسالة واتساب جاهزة عبر `wa.me/966555202585`، ويحفظ الطلب في `localStorage` (تجريبي)، ويعرض إشعار نجاح (toast).

## مصدر الصور
صور حقيقية للنشاط من ملف المشروع (مصدرها خرائط قوقل). تمت تصفيتها لاستبعاد أي وجوه واضحة أو ملصقات أسعار، والاكتفاء بصور الأظافر وأجواء الصالون.

## التشغيل محليًا
```bash
cd kala-spa
python3 -m http.server 8000
# ثم افتح http://localhost:8000
```

## رابط الموقع
سيُضاف بعد النشر على GitHub Pages.

## نتائج الاختبار
- تم التحقق محليًا: جميع الملفات والصور تُحمَّل بنجاح (HTTP 200)، وكل مرجع صورة موجود فعليًا.
- يُستكمل الفحص البصري على متصفح حقيقي (سطح مكتب 1440px + جوال 390px) قبل النشر.
